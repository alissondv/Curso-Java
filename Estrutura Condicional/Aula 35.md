# Estrutura condicional
## Conceito

Estrutura condicional:

>É uma estrutura de controle que permite definir que um certo bloco de comandos somente será executado dependendo de uma condição

## Sintaxe da estrutura condicional

~~~Javascript
if ( <condição> ) {
<comando 1>
<comando 2>
}
~~~

## Sintaxe da estrutura condicional

~~~Javascript
if ( <condição> ) {
<comando 1>
<comando 2>
}
else {
<comando 3>
<comando 4>
}
~~~
