# Expressões comparativas
**Operadores comparativos**
| Operador      | Significado |
| ------------- | ------------- |
| >             | maior  |
| <             | menor |  
| >=            | maior ou igual |  
| <=             | menor ou igual |  
| ==            | igual |  
| !=            | diferente |  


**Exemplos de expressões comparativas**
(suponha x igual a 5)

>X > 0 Resultado: V


>X == 3 Resultado: F


>10 <= 30 Resultado: V
